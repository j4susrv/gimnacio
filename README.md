1.Crear una carpeta para el proyecto

2.Clonar el repositorio
  Ejecutar en la terminal:
  git clone https://github.com/j4susrv/gimnacio

3.Cambiar a la rama master
  Después de clonar, asegurarse de estar en la rama correcta:
  git checkout master

4.Actualizar la rama master
  Para obtener los últimos cambios del repositorio remoto:
  git pull origin master
